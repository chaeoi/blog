一键安装BBR加速
```c
mkdir -p /opt/bbr && wget --no-check-certificate -O /opt/bbr/bbr.sh https://cdn.canghai.org/sh/bbr.sh && chmod 755 /opt/bbr/bbr.sh && bash /opt/bbr/bbr.sh
```
