#### jsDelivr
官网：[https://www.jsdelivr.com/](https://www.jsdelivr.com/)

支持`npm`仓库资源同步，`Github`资源同步。就目前体验而言，速度还不错。示例如下：
```c
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/lxgw-wenkai-webfont@1.6.0/lxgwwenkai-regular.css" />
```
#### UNPKG
官网：[https://unpkg.com/](https://unpkg.com/)

资源丰富，但国内速度不行。示例如下：
```c
<link rel="stylesheet" href="https://unpkg.com/lxgw-wenkai-webfont@1.6.0/lxgwwenkai-regular.css" />
```
#### Staticfile
官网：[https://staticfile.org/](https://staticfile.org/)

同步国外`CDNJS`源站，同时由国内开源贡献值提交其它有价值的库，由七牛云提供支持，速度理想。示例如下：
```c
<link rel="stylesheet" href="https://cdn.staticfile.org/lxgw-wenkai-webfont/1.6.0/lxgwwenkai-regular.css" />
```
#### eleme
官网：[https://npm.elemecdn.com/](https://npm.elemecdn.com/)、[https://github.elemecdn.com/](https://github.elemecdn.com/)

未公开访问，但已稳定运行多年。示例如下：
```c
<link rel="stylesheet" href="https://npm.elemecdn.com/lxgw-wenkai-webfont@1.6.0/lxgwwenkai-regular.css" />
```
```c
<link rel="stylesheet" href="https://github.elemecdn.com/lxgw-wenkai-webfont@1.6.0/lxgwwenkai-regular.css" />
```
